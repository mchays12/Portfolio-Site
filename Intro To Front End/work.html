Pokemon App

As part of this project, I developed a compact web application centered around Pokémon, utilizing HTML, CSS, and JavaScript. The application fetches data from an external Pokémon API and provides users with the ability to explore specific data attributes in depth. Users are initially presented with a list of Pokémon names. Upon selecting a Pokémon from the list, the application displays comprehensive information, including height, weight, types, abilities, and an accompanying image for that particular Pokémon.
